Projects

  • Ethical Principles in AI/ML Workflows

    This thesis project investigates the integration of ethical principles in AI/ML workflows, aiming to understand the current rate and challenges of implementing ethical standards in real-world applications. The study involves analyzing closed issues from AI/ML GitHub projects using large language models (LLMs) to detect ethical concerns and identify patterns. Additionally, interviews and surveys with developers and data scientists will provide insights into their awareness of ethical principles and the practical obstacles they face. This project contributes to the field by exploring both technical and human factors involved in promoting ethical AI development.

  • Wordle Reinforcement Learning Agent

    Collaborated on a team project to design and implement a reinforcement learning (RL) agent specialized in solving an variation of the popular game Wordle. The project involved creating multiple RL agents using different algorithms (such as Markov Decision Processes and Deep Reinforcement Learning) to explore optimal strategies. Our approach included injecting probabilistic noise to test the robustness of the agents, as well as experimenting with various game configurations to analyze changes in playstyle. The project showcased practical applications of RL in a constrained environment and provided insights into how AI can adapt strategies based on feedback.

  • Ghost Hunting

    Developed a multi-threaded C program to simulate a ghost-hunting mission inspired by the game Phasmophobia. This project utilized concepts like dynamic memory allocation, linked lists, and mutexes to simulate the interactions between ghost hunters and a ghost in a building with interconnected rooms. The system featured independent threads for each entity, enabling real-time actions such as evidence collection, communication, and movement. The hunters aimed to gather evidence to identify the ghost while managing fear levels, while the ghost roamed and left clues. The project emphasized resource management, synchronization, and concurrency, providing hands-on experience with systems-level programming.
